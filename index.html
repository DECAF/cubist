cubist.js

your content on a cube

 

 

Get started

  • download cubist.js from here
  • include cubist.min.js
    • with a good old <script/> element
    • or require('cubist') with a javascript module you like
  • include cubist.css in your HTML

Create

some cubist html with a maximum of 4 pages

<div id="cubist" class="cubist-stage">
    <section class="page">
    page 1
    </section>
    <section class="page">
    page 2
    </section>
    <section class="page">
    page 3
    </section>
    <section class="page">
    page 4
    </section>
</div>

the cubist instance in your javascript

var cubist = new Cubist(document.getElementById('cubist'));

and rotate that cube

cubist.goto(0);
cubist.next();
cubist.previous();
cubist.first();
cubist.last();

Use at your own risk!

Currently, cubist.js

  • is not tested
  • is not documented
  • does not have any old browser fallbacks
  • the "fit-into-your-browser" script used here is not part of cubist
  • some pre beta-ish thing
