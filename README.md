## Prerequisites
* install `compass` from [here](http://compass-style.org/)
* install `grunt-cli` from [here](https://github.com/gruntjs/grunt/wiki/Getting-started)
    
        $ npm install -g grunt-cli

## get started
* `$ npm install`
    
## build

    $ grunt
    
## watch

    $ grunt watch
